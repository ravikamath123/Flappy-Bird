# Flappy-Bird

A simple Flappy Bird game implemented using `pygame` module in python.

To install `pygame` module in
### Linux
```bash
sudo apt-get install python3-pygame
```
### Windows
```bash
pip install pygame
```
Press `SPACE` to control the movement of the bird in the game.
